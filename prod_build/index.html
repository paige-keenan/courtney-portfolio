Mall

  • Mall

    Mall is a retail-shopping app that allows users to build and shop at their own personalized malls right from their phones. Users can view curated galleries, view product info and even checkout from the app.

    branding
    web

    Copywriter:

    Keith Wasser

  • Mall's minimalist design prioritizes each brands personality over that of the app

  • A recency feed allows users to scroll through each stores latest additions

  • Window mode allows users to browse product galleries curated by each store

  • Users can edit items within the shopping cart or save them to return to later

  • A shared shopping cart allows users to checkout from every store at once
